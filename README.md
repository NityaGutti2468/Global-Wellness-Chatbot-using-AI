# 🧠 Global Wellness Chatbot using AI

An AI-powered wellness platform that delivers personalized support for mental, physical, and nutritional well-being through intelligent multilingual conversations. The application combines Natural Language Processing (NLP), transformer-based intent recognition, user analytics, and secure data management to provide an engaging and scalable wellness experience.

Built using **Python, Streamlit, Hugging Face Transformers, SQLite, and NLP technologies**, the system supports real-time interactions, user feedback analysis, multilingual accessibility, and administrative monitoring through an integrated dashboard.

## 🎯 Problem Statement

Many individuals struggle to access reliable wellness guidance tailored to their needs. Existing solutions often lack personalization, multilingual accessibility, and centralized wellness support.

The Global Wellness Chatbot addresses this challenge by providing an AI-powered conversational assistant capable of understanding user intents, delivering personalized recommendations, and supporting wellness engagement across multiple domains.
