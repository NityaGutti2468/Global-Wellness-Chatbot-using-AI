## 🧠 Global Wellness Chatbot using AI

An AI-powered wellness assistant designed to support **mental**, **physical**, and **nutritional health** through personalized, multilingual conversations 🌍💬. Built with **Python**, **Streamlit**, **HuggingFace Transformers**, and **SQLite**, the chatbot offers secure access, real-time feedback, and admin-level analytics — all deployed seamlessly on **Streamlit Cloud**.

---

### 🚀 Features

- 🔐 **User registration** with email, password, and preferred language  
- 🗃️ **SQLite database** for persistent storage of users, chats, and feedback  
- 🧩 **Modular architecture** for easy expansion — including:
  - Mood tracking 🧠  
  - Journaling prompts 📓  
  - Breathing exercises 🌬️  
- 📊 **Admin dashboard** with intent analytics, feedback review, and failed query tracking  
- 🌐 **Multilingual support** via deep-translator and alias-aware entity normalization  
- 🧠 **Transformer-based intent recognition** with safetensors model loading  
- 🧪 **CLI and Streamlit UI** for flexible interaction modes  
- ☁️ **Live deployment** on Streamlit Cloud with GitHub integration and Git LFS support
- 📈 Leverages AI-driven insights to deliver personalized wellness recommendations while maintaining scalable, user-centric system design.
